Professional Plots
Open-source at MATLAB File Exchange

This toolbox grew out of a personal project of mine. I was working on a script that would automate the process of setting a few "good" properties for my plots. I was also struggling with hex codes and would be jumping on and off of google to get the right colors (#relatable right?). So I made a color palette and a scheme with it that would make it easy to pick colors.

For eg: If I want red, then I can use PS.Red1 if I want a darker shade, I can just increase the number PS.Red5 ~Phewww~

The toolbox now has over 1600 downloads!!! (BOOOM!)
